# 测字算事 - PC后台管理系统需求

## 项目概述
- **项目名称**: 测字算事 PC管理后台
- **功能**: 管理测字Prompt配置、查看历史记录、选择LLM模型
- **目标用户**: 管理员

## 功能需求

### 1. Prompt 配置管理
- 设置测字主Prompt模板，支持变量替换
- 可用变量: `{char}` (字), `{question}` (问题), `{direction}` (方位), `{time}` (时辰)
- 预览Prompt效果
- 保存/加载配置

### 2. 测字历史记录
- 查看所有历史测字记录
- 每条记录包含:
  - 用户输入 (字、问题、方位、时辰)
  - 发送给LLM的完整Prompt
  - LLM返回的原始结果
  - 前端展示的处理结果
- 支持搜索和筛选

### 3. LLM模型管理
- 支持多个LLM模型 (MiniMax, OpenAI, Anthropic等)
- 可选择默认使用的模型
- 模型配置 (API Key, Endpoint等)

## 技术方案

### 前端
- 独立HTML页面 (admin.html)
- 简洁的管理界面
- 使用 fetch API 与后端通信

### 后端
- 新增 `/api/admin/*` 路由
- 使用 SQLite 存储配置和历史记录
- 简单易部署

### 数据存储
- SQLite 数据库 (本地文件)
- 表: configs, history, models

## 页面设计

### 布局
- 左侧导航栏
- 右侧主内容区
- 三个Tab: Prompt配置 | 历史记录 | 模型管理

### 颜色主题
- 深色主题，符合测字传统风格
- 主色: #1a1a2e (深蓝黑)
- 辅色: #d4a574 (金色)
