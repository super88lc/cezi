## 快速部署

### 1. 安装依赖

```bash
cd zi-cesuan
pip install -r requirements.txt
```

### 2. 配置环境变量

```bash
cp .env.example .env
# 编辑 .env 填入真实配置
```

### 3. 运行

```bash
# 开发模式
python app.py

# 访问 http://localhost:5000
```

### 4. 部署到Vercel

```bash
vercel --prod
```

## 测字算法

- **拆字法**: 笔画、结构分析
- **五行判定**: 笔画数+偏旁部首
- **梅花易数**: 时间+字象起卦
- **吉凶判断**: 笔画数对应吉凶

## 环境配置

### 百度OCR

1. 访问 [百度智能云](https://console.bce.baidu.com/)
2. 注册账号 → 开通"文字识别"
3. 创建应用获取 API Key

### 微信支付

1. 开通[微信小程序](https://mp.weixin.qq.com/)
2. 开通[微信支付商户](https://pay.weixin.qq.com/)
3. 配置商户号和API密钥
